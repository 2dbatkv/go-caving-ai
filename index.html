Caving Software and AI

Interested in how cave survey software is created? Interested in how AI can be used to generate applications and analyze cave survey data?

Join me (Aaron Bird) and Philip Schuchardt for a discussion on software design process, vibe coding with AI, preferred coding and scripting languages, and anything else that comes up.

Sounds really nerdy? Yup. Pretty much is. But now is the time when this kind of nerdy is available to everyone. Anyone (everyone!) with good questions, some patience, a computer, and internet connection can make use of these modern technologies. And, yes, that means you, too.
